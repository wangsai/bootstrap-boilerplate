Bootstrap Boilerplate
***

## Quick Start

Check out the repository, install Node.js and then:

```sh
cd bootstrap-boilerplate
sudo npm -g i grunt-cli bower
npm i && bower install
grunt watch
```
Finally, it opens in your browser.

Happy hacking!
